# Screenshots Directory

This directory contains visual documentation of the Pyongyang University Network Monitoring Infrastructure.

## Contents

Screenshots are organized by category:

1. **OpenNMS Dashboard** - Monitoring platform overview
2. **Node List** - All monitored devices and services
3. **Services** - Individual service interfaces
4. **Network Topology** - Visual network representation
5. **Campus Web Servers** - Remote location portals
6. **Monitoring Details** - Detailed monitoring views
7. **Infrastructure** - Container and system views

## File Naming Convention

- `01_opennms_dashboard_*.png` - OpenNMS main views
- `02_nodes_*.png` - Node lists and details
- `03_services_*.png` - Service interfaces
- `04_topology_*.png` - Network topology views
- `05_campuses_*.png` - Campus-specific views
- `06_monitoring_*.png` - Monitoring details
- `07_infrastructure_*.png` - System infrastructure

## How to Use

1. Review SCREENSHOT_GUIDE.md for detailed checklist
2. Access services using QUICK_ACCESS.txt
3. Capture screenshots following naming convention
4. Place files in this directory

## Screenshot Tool Recommendations

**Linux:**
- gnome-screenshot (GNOME)
- spectacle (KDE)
- flameshot (Cross-platform)

**Browser:**
- Firefox: Built-in screenshot tool
- Chrome: Developer tools screenshot
